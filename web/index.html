📥 ダンプ / 解析

接続中のモジュール(0x50)を読み出して解析表示します。SFP は SFF-8472(128B)、QSFP/QSFP28 は SFF-8636(上位ページ 00h を含む 256B)として解析します。.bin で手元に保存できます。

📡 DWDM 波長設定 tunable

    
チューナブル SFP の ITU グリッドチャンネルを設定します。0x51 のページを 2 に切替え、freq×20 を レジスタ 146/147 に書き込みます。SFP(Bus0) 専用機能です（QSFP には 0x51 が無いため非対応）。

🔧 Avago ISP 書換 device-avago-isp 専用ボード

    

      device-avago-isp ボード（FW v2.x）で、Avago/Broadcom 内蔵 ATmega328 を
      ISP(SPI シリアルプログラミング) 経由で直接読み書きします。I2C(0x50)では書き込めない個体でも、
      内部 EEPROM を直接書き換えられます。
    

    

      ⚠️ 専用ファーム(FW v2.x)と専用配線が必須です。
      配線: MOSI=pin4/GP4, SCK=pin5/GP5, MISO=pin9/GP9, /RESET=pin15/GP15, VCC=pin16/3V3, GND=pin1。

      ISP 中は I2C(他タブ)は使えません。書込前に必ず「③ EEPROM バックアップ」で 1KB を保存してください。
      Chip Erase は行いません（フラッシュ＝ファームを消さない）。
    


    

    
① ISP 接続 / デバイス確認

    
/RESET を Low にして Programming Enable を送り、signature・lock・fuse を読み出します。終了時は必ず「ISP 終了」で RESET を解放してください。

    

      
      
      
    

    
—


    

    
② EEPROM バックアップ (1KB 全吸出し)

    
内部 EEPROM 1024 バイトを丸ごと吸い出して .bin 保存します。書込前に必ず取得してください。

    

      
      
    


    

    
③ ファーム(フラッシュ)ダンプ (32KB)

    
内蔵 ATmega328 のフラッシュ 32768 バイト（ファーム本体）を ISP で吸い出して .bin 保存します。読み出しのみ・無改変です。

    

      
      
    


    

    
④ A0 クローン書込 (内部 EEPROM 0x300–0x37F = 0x50 ページ)

    

      選んだ EEPROM 像を内部 EEPROM 0x300 以降へ ISP で書き込みます。チェックサム(0x3F/0x5F)は自動再計算、
      実 SN(0x44–0x53)・製造日(0x54–0x5B)は既定で維持、書込後にベリファイします。
      別バスからのコピーは ISP 開始前（I2C 有効時）に行ってください。
