🔓 Avago 隠し Mailbox 検証用

    

      Avago/Broadcom 系モジュールの hidden mailbox（B0 ページ = 0x58）経由で内部 EEPROM を読み書きします。
      コマンドは 0x76 から 5 バイトを 1 トランザクションで送ります（ページ境界 0x78 を跨ぐため通常の分割書込は不可）。
    

    

      ⚠️ 低レベル検証ツールです。内部 EEPROM を直接書き換えます。前提として内部 EEPROM 0x01FF == 0x55 が必要で、
      ファームが起動時にこれを読み 0x0336 に入れ、0x55 のときだけ hidden command handler を有効化します。
      外部から 0x58 が見えない個体ではホスト側から叩けません。
